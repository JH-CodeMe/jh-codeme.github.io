파이썬 Python

``` python
print("CodeMe")
```

파이썬을 조금이라도 해보신 분이라면 이 코드가 어떻게 동작하는지는 아주 쉽게 알 수 있을겁니다.
print 함수는 콘솔창에 괄호 안 문자열을 출력합니다.
콘솔창에 CodeMe가 출력될 것입니다.

파이썬을 처음 해보시는 분이라면 겁먹지 말고 집에서 강의를 보신다면 먼저 파이썬을 설치해주세요.
죽전고등학교 컴퓨터실에는 파이썬이 설치되어있기 때문에 설치할 필요가 없습니다.
파이썬을 설치하기 위해서는 먼저 https://www.python.org/downloads/ 이 링크에서 파이썬을 다운로드 받고 실행시켜서 설치해줍니다.

그리고 키보드에서 윈도우키+q 로 검색 메뉴를 열고 python을 검색하고 python shell을 실행합니다.

이런식으로 출력되면 정상적으로 설치가 완료된 것입니다.(버전은 차이가 있을 수 있지만 큰 문제는 없습니다)

그럼 반 단계 정도 올라간 코드를 보겠습니다.
# 1. 변수
``` python
myvar = "CodeMe"
print(myvar)
```

결과는 똑같이 CodeMe입니다. 여기서 사용된 개념이 변수인데요. 변수는 계속 변하는 값으로 코드 중간에 계속 값을 변경할 수 있습니다. 파이썬에서는 변수를 처음 사용하면 자동으로 변수가 만들어집니다. 
``` python
myvar = "CodeMe"
```
이 부분에서 변수를 myvar이라는 변수를 처음으로 사용했는데 위에서 말한대로 그전에 myvar이라는 변수가 만들어지고(물론 코드에서는 따로 변수를 만들지 않습니다. 자동으로 파이썬에서 해줍니다.)
여기서 변수를 만든다고 했는데 우리는 이를 변수를 선언한다고 합니다.

그리고 변수를 사용할 때는 변수 그대로 입력해주면 됩니다. 물론 변수만 딸랑 써놓으면 아무일도 일어나지 않으니 print 함수 괄호 안에 써주면 괄호 안의 내용을 출력하는 print 함수가 괄호안의 내용인 myvar의 값을 출력하겠죠?

``` python
myvar = "CodeMe"
print(myvar)
myvar = "Hello World!"
print(myvar)
```

중간에 값을 변경해봤습니다. 결과를 미리 알려드리면
CodeMe
Hello World!
로 출력됩니다.

처음에 변수가 처음 사용되었으므로 자동으로 선언되면서 "CodeMe"라는 문자열 값을 저장하고 print 함수가 실행된 뒤 myvar 변수가 이미 있으므로 선언하지 않고 "Hello World!"라는 문자열 값으로 값이 변경됩니다. 그리고 print 함수가 실행되어서 위와 같은 결과가 나오는 것입니다.
혹시라도 변수의 개념이 이해가 잘 안된다면 변수를 상자라고 생각하고 그 안에 들어가는 값을 물건이라고 생각하면 이해가 쉽습니다. 상자안에 물건을 바꾸고 그 물건을 쓸 수도 있고요.

# 2. 연산자
연산자를 설명하기 이전에 타입에 대해 간단히 알아봅시다. 타입은 자주 사용하는 타입 위주로 설명하면 int(정수), float(실수), string(문자열), bool(참, 거짓), list(이후에 다룸), dict(이후에 다룸) 등이 있는데요. 여기서 특징은 문자열은 양끝에 "을 붙입니다. "CodeMe" 와 같은 식으로요.
## 2.1. 산술 연산자
### +
연산자 앞의 값에서 뒤의 값을 더한 값을 반환합니다.
이는 숫자(int, float 등)일 때 적용되고 문자열일 경우의 연산자 앞의 값의 문자열에 뒤의 값의 문자열을 합친 값을 반환합니다.
### -
연산자 앞의 값에서 뒤의 값을 뺀 값을 반환합니다.
### *
연산자 앞의 값에서 뒤의 값을 곱한 값을 반환합니다.
### /
연산자 앞의 값에서 뒤의 값을 나눈 값을 반환합니다.
### %
연산자 앞의 값에서 뒤의 값을 나눈 나머지를 반환합니다.
### **
연산자 앞의 값에 뒤의 값만큼 거듭제곱한 값을 반환합니다.

ex)
``` python
num = 2
result = num + 3
print(result)
```
num 변수에 2를 할당(값을 넣는다고 생각하시면 됩니다)하고 result 라는 변수를 선언함과 동시에 num에 3을 더한 값을 할당한다(즉 5) 그리고 result의 값을 출력한다.
5가 출력되겠죠? 이런식으로 산술 연산자를 활용하시면 됩니다.

## 2.2. 할당 연산자
### =
연산자 앞의 변수에 뒤의 값을 할당합니다. 즉 변수에 값을 할당할 때 쓰이는 아주 기본적인 연산자입니다.
### +=
연산자 앞의 변수의 값에 뒤의 값을 더한 값을 앞의 변수에 할당합니다.
ex)
``` python
num = 1
num += 3
print(num)
```
3이 출력됩니다.
### -=
위 연산자와 비슷하지만 앞의 변수의 값에 뒤의 값을 뺀 값을 앞의 변수에 할당합니다.
### *=, /=, **=
위 연산자들과 비슷한 맥락으로 이해하시면 될 것 같습니다.

## 2.3. 비교 연산자
### == 
연산자 앞의 값과 뒤의 값이 같으면 True(참)를 반환하고 다르면 False(거짓)를 반환합니다. 이는 조건문이나 반복문 등에서 많이 쓰이는데 아래에서 다루겠습니다.
### !=
위의 연산자와 반대입니다. 연산자 앞의 값과 뒤의 값이 다르면 True를 반환하고 같으면 False를 반환합니다. 이것도 조건문, 반복문 등에서 많이 쓰입니다.
### >
연산자 앞의 값이 뒤의 값보다 크면 True를 반환하고 작거나 같으면(크지 않으면) False를 반환합니다.
### <
연산자 앞의 값이 뒤의 값보다 작으면 True를 반환하고 크거나 같으면(작지 않으면) False를 반환합니다.
### >=, <=
위의 부등호 연산자들과 비슷하지만 이 연산자는 크거나 같다 또는 작거나 같다 를 나타냅니다.

## 2.4. 논리 연산자
논리 연산자는 기본적으로 앞, 뒤 값(피연산자)가 bool 형식의 값, 즉 True나 False여야 합니다. 그래서 비교 연산자로 bool 형식의 값을 얻은 후 그걸 논리 연산자에 대입해서 많이 사용합니다.
### and
앞의 값과 뒤의 값이 둘다 True일때만 True를 반환하고 나머지는 모두 False를 반환합니다.
### or
앞의 값과 뒤의 값 중 하나라도 True라면 True를 반환하고 모두 False일 때만 False 를 반환합니다.
### not
이 연산자는 앞의 값이 없습니다. 뒤의 값만 있으며 뒤의 값을 반전합니다.
``` python
hello = 1
hi = 2
three = 3

mybool = hello == hi and three == three
print(mybool)
```
hello == hi 의 연산 결과는 False가 나오고 three == 3의 연산 결과는 True가 나오는데 and 연산자이므로 둘다 True 가 아니기 때문에 False를 출력합니다.

이쯤돼서 수학에 연산 순서가 있듯이 연산자의 연산 순서를 간단하게 알아보겠습니다. 아래로 갈수록 점점 더 늦게 연산됩니다.
**
+, -
*, /, %
비교 연산자
논리 연산자

원래는 더 많지만 간단히 지금까지 배운 연산자만 정리해보면 이렇습니다.

# 3. 함수
함수라는 이름에 겁먹지 마세요
함수는 어떠어떠한 일을 하기로 약속하고 그 약속해 놓은 함수를 이후에 여러번 사용할 수 있습니다.
함수의 선언은 다음과 같이 이루어집니다
``` python
def myfunc(mynum, myvar):
	return mynum + myvar
print(myfunc(1, 4))
```
출력: 5

myfunc 아래 내용은 들여쓰기를 해줘야하는데 일반적으로 탭 키를 사용합니다. 탭 키와 스페이스 바를 혼용하지만 않으면 됩니다.(혼용할 경우 에러 발생)
myfunc라는 함수를 선언하면서 함수의 매개변수로 괄호안의 첫번쨰 값을 받아 mynum이라는 변수로 함수내에서만 사용하고 두번째 값을 myvar이라는 변수로 함수내에서만 사용합니다. 즉 함수 바깥에는 mynum, myvar이라는 변수는 없습니다.
return은 반환한다는 의미로 위 코드에서는 mynum 과 myvar 을 합한 값을 반환합니다. 그러면 변수가 값을 갖듯이 myfunc라는 함수는 자연스럽게 첫번째 매개변수 + 두번째 매개변수 이라는 값을 갖는 것이죠.
반면 return 값이 없는 함수도 있는데 그 경우 반환하는 값은 없고 함수의 내용만 실행합니다.
``` python
def myprint(string):
	print(string)
myprint("Hello World!")
```
출력: Hello World!

위는 return 값이 없는 함수입니다.

사실 함수를 사용하지 않고도 프로그래밍이 어느정도는 가능합니다. 하지만 반복적인 일을 할 때 코드의 길이를 줄이고 가독성에도 좋고 궁극적으로 코딩을 하는 사람의 수고가 덜어지니 사용하는 편이 좋습니다. 또 재귀함수라는 함수를 사용하지 않고선 구현이 힘든 기능도 있습니다.
여기서 재귀함수를 간단하게 알아보면 함수 안에서 함수 자신을 호출하는 함수입니다.
# 4. 리스트, 딕셔너리
아까 파이썬의 타입에 대해 간단하게 설명할 때 list와 dict가 나왔습니다. list는 말그대로 리스트이고 dict는 딕셔너리를 뜻하는데요 각각 알아보겠습니다
## 4.1. 리스트
리스트의 선언
``` python
mylist = []
numlist = [1, 2, 5, 1]
print(numlist[1])
print(numlist.size())
```
출력:  
2  
4

첫줄은 mylist라는 빈 리스트를 선언하고 두번째 줄은 1, 2, 5, 1이라는 값이 담긴 리스트를 선언합니다.
이 요소들에 접근할때는 인덱스라는 것을 사용하는데 인덱스는 0부터 시작하기 때문에 첫번째 값은 0, 두번째 값은 1, 세번째 값은 2 ... 이런식입니다.  
그렇기 때문에 세번째 줄에 numlist[1]은 numlist의 두번째 값을 반환합니다.
리스트가 갖고 있는 함수들도 있는데 몇가지 정리해보면  
size()   리스트의 크기를 나타냅니다(여기서 주의할 점은 인덱스는 0부터 시작하지만 size는 말그대로 크기를 나타냅니다)
append(value)   리스트 끝에 값을 추가합니다.  
따라서 numlist.size() 함수는 4를 반환하고 그 값이 print 함수에 전달되어 4가 출력되었습니다.
## 4.2. 딕셔너리
``` python
mydict = {"first": "hi", "second": "hello"}
print(mydict["first"])
```
딕셔너리는 기본적으로 key: value의 형태를 가집니다. key: value 의 형태가 리스트처럼 여러개가 있는 형태입니다.  
딕셔너리는 인덱스대신 [key] 로 값에 접근합니다. 값 추가나 변경은  
mydict[key] = value  
여기서 key가 없는 key면 새로운 key: value 쌍이 추가되고 있는 key면 value로 값이 변경됩니다.
## 4.3. 튜플
수학에서 집합과 같습니다. 순서는 무조건 오름차순이고 값은 숫자만 들어갈 수 있습니다. 타입은 set이지만 튜플이라고 많이 불러서 여기서는 튜플로 설명했습니다.
``` python
mytuple = {1, 4, 5, 1, 2}
print(mytuple)
mytuple.add(3)
print(mytuple)
```
출력:  
{1, 2, 4, 5}  
{1, 2, 3, 4, 5}

첫번째 줄에선 튜플이기 때문에 겹치는 숫자는 없어지고 오름차순으로 정렬됩니다.
그리고 출력된 후 2를 요소로 추가한 후 다시 정렬되어서 출력됩니다.
# 5. 조건문
## if문
``` python
if True:
	print("it is true")
```
출력: it is true

기본적으로 if문은 뒤의 값이 True 이면 아래 내용을 실행합니다. 괄호 안의 값은 변수나 함수가 될 수도 있고 연산자가 있는 값이 될 수도 있습니다. 다른 예를 들어보겠습니다.
``` python
if1 == 1:
	print(it is true)
```
출력: it is true

``` python
isOn = False
if isOn:
	print("hi")
```
출력: 없음
``` python
num = 2
isOn = False
if num == 2 or isOn:
	print("executed")
```
출력: executed

## elif 문
여러개의 if 문을 이어서 사용할 때 elif 문을 사용합니다. 예시는 밑의 else문과 같이 알아보겠습니다
## else 문
if-elif 문 마지막에 사용하며 이러한 조건들이 모두 False 이면 실행됩니다.

ex)
``` python
mynum = 2
num = 4
if mynum == 3:
	print("hi")
elif num == 2:
	print("hi")
else:
	print("this is else")
```
출력: this is else

위의 두 print 함수들은 조건에 맞지 않아 실행되지 않고 가장 밑의 else 문만 실행이 된 것을 확인할 수 있습니다.
# 6. 반복문
## 6.1. while
``` python
while True:
	print(hi)
```
출력:  
hi  
hi  
hi  
hi  
.... 무한반복

while 문은 뒤의 값이 True일 때 계속 반복합니다.
## 6.2. for
다른 언어를 해보신분들이라면 파이썬의 for 문은 색다르게 느껴지실 겁니다. 굉장히 편한 형태이지만 그만큼 다른 언어와 문법이 다르다는 단점을 갖고 있습니다.  
기본적으로 파이썬의 for 문은 for in 형태입니다. in 뒤에는 리스트, 문자열, 튜플이 오고 in 앞에는 변수가 옵니다. for 문은 in 뒤에 있는 리스트의 크기만큼 반복을 수행합니다. 예시를 들어보면
``` python
mylist = [1, 2, 4, 5, 10, 52]
for i in mylist:
	print(i)
```
출력:  
1  
2  
4  
5  
10  
52  

리스트의 크기가 6이므로 6번 수행했습니다. 수행 할 때마다 i의 값이 mylist의 0번 인덱스부터 하나씩 올라갑니다.  
하지만 반복을 정해진 수만큼 수행하는 경우도 있을겁니다. 그렇다고 그때마다 정해진 크기의 리스트를 생성할 수도 없습니다. 하지만 파이썬에는  range라는 함수가 있습니다.  
``` python
range(10)
```
위 코드는 0 부터 10 미만까지를 포함하는 객체를 생성합니다. 여기서 객체는 설명하면 복잡하니 그냥 0부터 10 미만의 정수가 들어있는 리스트라고 생각하시면 됩니다.

``` python
for i in range(5):
	print("hi")
```
출력:  
hi  
hi  
hi  
hi  
hi  
위처럼 5번 반복합니다.

# 7. 입출력 함수, 그리고 개행문자
## 7.1. 입력 함수
``` python
text = input()
print(text)
```
위의 코드는 바로 출력이 되지 않고 먼저 입력을 받습니다. 그리고 입력한 문자열을 출력하는 코드입니다.  
``` python
text = input("입력해주세요: ")
print(text)
```
위의 코드는 입력해주세요: 라는 문자열이 출력된 후 바로 옆에 입력을 받습니다.  
결과는 똑같이 입력한 문자열이 출력될 것입니다.

## 7.2 출력 함수와 개행
출력 함수는 여러분들이 다 알고 있듯이 print 함수가 있습니다. print 함수는 괄호 안의 내용을 콘솔창에 출력해주죠. 
``` python
print("Hello")
print("World")
```
출력:  
Hello  
World  
  
위와 같이 출력되는데요. 보시다시피 콘솔에 Hello가 출력된 뒤 자동으로 줄이 바뀌어서 World가 출력되었죠. 물론 처음에는 편할 수 있지만 개행을 하지 않고 계속 한줄에 출력하고 싶을 때가 있을겁니다. 코드를 먼저 보여드리면
``` python
print("Hello ", end="")
print("World")
print("next line")
```
출력:  
Hello World  
next line  
  
이렇게 출력됩니다. 여기서 end는 Hello를 출력한 뒤 끝에 어떤 문자를 붙일 것인지 정하는 매개변수인데 ""(빈 문자열) 로 정해서 아무것도 출력되지 않고 한줄에 그대로 출력되게 합니다  
그러면 왜 end를 지정하지 않으면 줄이 넘어가는걸까요?  
답은 기본값에 있습니다. end 매개변수의 기본값은 \n으로 개행문자입니다. 개행문자란 커서를 다음 줄로 넘겨주는 역할을 합니다. 예를 들면
``` python
print("Hello\nWorld")
```
출력:  
Hello  
World  
이런식으로 개행 문자가 문자열 사이에 있으면 개행이 됩니다.

# 8. 이외
## 8.1. 주석
주석은 코드안에서 설명을하는 역할을 합니다. 주석 안의 내용은 코드에 영향을 주지 않습니다. 그래서 대부분 코드에 대한 설명을 적는 편입니다.
``` python
print("Hello, World!") # Hello World는 모든 언어의 시작입니다
```
이런식으로 코드에 영향이 가지 않게 설명을 할 수 있습니다. 위의 주석은 한줄 주석이고 주석을 여러줄에 걸쳐서 쓸때는 하나 하나 #을 붙여주지 않고 아래와 같이 사용합니다.
``` python
print("Hello, World!")
"""
여러줄
주석
입니다
"""
```
큰따옴표가 아닌 작은따옴표도 가능합니다.
## 8.2. 클래스
클래스는 자동차가 있다고 하면 자동차의 설계도입니다. 그리고 객체는 설계도를 이용해 만든 자동차로 비유할 수 있습니다. 
``` python
class Car():
	engine = "V8"
	color = "black"
	cartype = "SUV"
car = Car()
print(car.engine, car.color, car.cartype)
```
출력:  
V8 black SUV  
하지만 객체안의 변수 값이 바뀐다고 클래스 자체의 변수나 다른 객체의 변수가 바뀌진 않습니다. 예를 들어보겠습니다.
``` python
class Car():
	engine = "V8"
	color = "black"
	cartype = "SUV"
car = Car()
mycar = Car()
mycar.color = "red"
print(car.color)
print(mycar.color)
```
출력:  
black  
red  
mycar 객체의 color 변수값을 변경해주었지만 car 객체의 변수값은 변하지 않았습니다.  
## 8.3 라이브러리
import를 통해 라이브러리를 추가할 수 있습니다. 먼저 파이썬 내장 라이브러리인 random 을 사용해보겠습니다. 내장 라이브러리이므로 설치가 필요없습니다.
``` python
import random
print(random.random())
```
출력: 0.3605802633142553  (저의 경우)
위와 같이 0 이상 1미만의 숫자가 말그대로 랜덤으로 나올 것입니다. 이것을 난수라고 합니다.
``` python
import random
print(random.randrange(10))
```
출력: 5 (저의 경우)
아까 배웠던 range 함수를 생각하시면 편합니다. 0 이상 10 미만의 정수를 랜덤으로 뽑아서 반환합니다.
